## ✨ 主要功能

- 🔧 系统管理
- 📺 监控管理
- 🔨 代码生成
- 🚙 车辆环保信息管理 
- 📝 OCR识别
- 📱 微信端服务

## 💻 开发环境

- Node.js >= 16
- yarn >= 1.22

## 📁 项目结构
```bash
├── bin                     # 启动脚本
├── src                     # 源代码
│   ├── api                # API接口
│   ├── assets             # 静态资源
│   ├── components         # 公共组件
│   ├── layout             # 布局组件
│   ├── router            # 路由配置
│   ├── store             # 状态管理
│   ├── utils             # 工具函数
│   └── views             # 页面组件
├── public                 # 静态资源
└── package.json          # 项目依赖
```
